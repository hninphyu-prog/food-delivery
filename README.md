# FoodAndMe - Food Delivery Service System
## Project Overview

FoodAndMe is a team project developed as a web-based food delivery service system.

The system connects customers, restaurants, riders, and administrators through a centralized platform. It allows customers to order food from nearby restaurants, while restaurants can manage menus and orders, and riders can handle delivery operations.

Through this project, our team gained practical experience in web application development, database design, system implementation, and team collaboration.


## Main Features

### Customer
- Account registration and authentication
- Browse restaurants and menus
- Add items to cart and place orders
- Track order status
- View order history
- Submit reviews and ratings

### Restaurant
- Manage restaurant profile
- Create and update menu items
- Handle customer orders
- Monitor sales and settlements

### Rider
- Manage assigned deliveries
- Update delivery status
- Track earnings and payout information

### Admin
- Monitor overall system activities
- Manage users, restaurants, and riders
- Generate settlement reports
- Export financial records in CSV/PDF format
